## Installation

Depends on [BaseLib](https://github.com/Alchyr/BaseLib-StS2/releases)

Extract the .zip from the [latest release](https://github.com/Ind-E/CorpseExplosionStS2/releases/latest) into your Slay the Spire 2 mods folder
- Windows and Linux: `Steam/steamapps/common/Slay the Spire 2/mods/`
- Mac: `SlayTheSpire2.app/Contents/MacOS/mods`
